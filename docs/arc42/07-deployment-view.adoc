= 7. Deployment View

Status: blocked/planned. No confirmed deployment topology exists yet.

== 7.1 Deployment Scope

Deployment View maps C4 containers to runtime nodes and physical or virtual placement. It does not redefine C2 containers.

Current confirmed rule:

* HSM and TPM capabilities are inside the solution boundary when product-owned or product-managed.
* Physical HSM/TPM placement belongs in this chapter once deployment decisions exist.
