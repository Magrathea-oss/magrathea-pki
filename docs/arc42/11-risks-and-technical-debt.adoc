| Manifest signing and trusted distribution unresolved
| risk
| A content hash is only tamper-evident when compared with a trusted expected ID from `profiles/profiles.manifest.yaml`; an attacker who can change both profile YAML and manifest can bypass detection unless manifest distribution and protection are trustworthy.
| Decide the first-release trust/distribution model and when to require `profiles.manifest.sig`, software signatures in development, HSM signatures in production, Git signed commits, append-only audit, or transparency logs.
| security architect / compliance officer / operations
