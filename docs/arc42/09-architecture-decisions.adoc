= 9. Architecture Decisions

Status: implementation-informed.

ADR bodies are owned by adr-writer and are not generated in this chapter. This chapter indexes existing ADRs and highlights missing decision records.

== 9.1 ADR Index

[cols="1,1,3",options="header"]
|===
| ADR | Status | Notes

| link:../adr/0001-use-content-addressed-yaml-for-certificate-profiles.md[ADR 0001 — Use content-addressed YAML for certificate profiles]
| Accepted
| Certificate profiles are immutable, content-addressed YAML resources. Profile IDs use a versioned semantic canonical hash such as `x509-profile-v1-sha256-<digest>`.
|===
